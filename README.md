# pagenote
-  基于node+express开发
- 用来代替wiki的个人文档系统
 
#特点
 * 支持粘贴截图(暂不支持文件)
 * 支持粘贴doc (暂不支持图文混合)
 
#安装
 * 安装node,安装express,jade (如果提示依赖请按提示操作就行了)
 * git clone https://github.com/youwi/pagenote.git
 * cd pagenote
 * node bin/www
 
# 文件说明
- public 前端html加js
- routes 后端接口加js
- views  jade使用的,暂时没有使用到
- doc 保存的笔记,测试用的

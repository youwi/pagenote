wiki平台笔记

 

Wiki目录层次组织

 

我的主页---->我的子页面----->下级页面---->下级页面

html     html           html         html     

 

目录结构:

│  我的主页.html

│  我的主页.zip

└─我的主页

    │  我的子页面.html

    └─我的子页面

        │  下级页面.html

        └─下级页面*

 

 

zip文件为历史文件,保存为编辑过的文件,可以为多个.

如:我的主页.zip.2015

zip文件内的内容以日期为后缀名如:我的主页.html.20140509

 

 

 

 

 

 

Wiki的web接口:

/我的主页.我的子页面 

访问wiki(带有模板,有编辑功能)

/我的主页/我的子页面.html

直接访问wiki(没有编辑功能)---静态文件(服务器压力小)

(1,nginx实现;2,拦截器实现,2种可以并存.)

/wikiSave

直接保存的接口(把wiki按字符串方式上传)

/wikiUpload

上传文件的接口(可以上传图片或wiki文档(文件)

doc文件自动转换格式.

/wikiEdit

进入编辑页面(有变体,如下:)

/wikiDelete

删除页面(有变体,如下:)

/wikiRename

重命名(有变体,如下:)

/wikiTag

设置标记,过滤用的,如分组功能(有变体,如下:)

/wikiDir

获取目录,展示用的(有变体,如下:)

/wikiLock

加锁

/wikiAdd

加新页面

/wikiMove

移动页面

/wikiFork

建立连接地址(如  /abc  指到 /我的主页.我的子页面

/wikiSearch

搜索页面和子页面

/wikiHis

历史页面

 

 

 

变体:

 

我的主页.我的子页面?Edit

进入编辑页面

我的主页.我的子页面?Reneme

重命名

我的主页.我的子页面?Delete

删除

我的主页.我的子页面?Tag

设置过滤标记

我的主页.我的子页面?Dir

获取子目录,用于下级显示,查看图片文件这类的

我的主页.我的子页面?Lock

加锁,限制编辑

我的主页.我的子页面?Add

添加新页面

我的主页.我的子页面?Move

移动页面

我的主页.我的子页面?Fork

建立连接地址

我的主页.我的子页面?Search

搜索页面和子页面

我的主页.我的子页面?Quote

查找引用(如主页引用了子页面) 

我的主页.我的子页面?His

查看历史页面.

 

 

   

 

wiki命名规定:

wiki名字不能带有点号!

wiki名字不能为3英文(如txt,html这类名字,因为默认当静态资源)
